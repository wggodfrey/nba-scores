# nba-scores

## SETUP

`git clone https://wggodfrey@bitbucket.org/wggodfrey/nba-scores.git`

`cd nba-scores`

`npm install`

`npm run build`

## RUN IN BROWSER

`cd public/`

`open ./index.html`

## POPULATE DATA

>> copy + paste from https://s3.amazonaws.com/wgg-random/kubit/nba.csv into text area
